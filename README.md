# Doctor App

Flutter app use Clean Architecture with Cubit, this app help people to book doctors' clinics, with amazing UI.

## Features

- Splash Screen
- Onboarding Screen 
- Login Screen
- Sign Up Screen
- Main Screen
- 

## Used

- Retrofit
- Dio
- Bloc (Cubit)
- Freezd
- json_annotation
- Get It
- Localization
- Native Splash
- cached_network_image
- shared_preferences
- firebase_core
- flutter_svg
